# DIRECTRICES VISUALES Y CORE ESTÉTICO

## 1. Estética y Filosofía
- **ADN Visual:** Inspirado en el Top 1% de Silicon Valley (estética Linear, Paco Coursey, Rauno Freiberg)
- **Dark Mode Profundo:** El fondo debe ser negro puro (#000000)
- **Interfaz Primaria:** La interfaz principal es el texto, apoyado en el uso extremo del espacio negativo

## 2. Prohibiciones Críticas (Cero Riesgos)
- **CERO sombras baratas (drop-shadows).** La separación visual se logra con bordes sutiles y cambios de color de fondo
- **CERO imágenes genéricas de stock o capturas de pantalla aburridas.** Usa tipografía gigante y métricas abstractas desenfocadas (ej. "-100%", "€120k+") de fondo

## 3. Micro-interacciones y UI
- **Glassmorphism:** Las tarjetas usan un efecto táctil sutil. Borde `border-white/10` y fondo `bg-[#141414]` al pasar el ratón
- **Menú de Comandos (⌘K):** Esencial para transmitir el aura de "Builder". Navegación por teclado sin tocar el ratón
- **Animaciones:** Tienen que ser microscópicas y perfectas, evitando cargas pesadas
